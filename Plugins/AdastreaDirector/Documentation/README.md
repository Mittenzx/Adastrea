# Adastrea Director Plugin Documentation

This directory contains documentation specific to the Unreal Engine plugin component of Adastrea Director, organized following UE plugin standards.

## Directory Structure

### `/features`
Documentation for specific plugin features including:
- Status indicators
- Tabbed UI interface
- RAG integration
- UE Python API
- Blueprint integration

### `/guides`
User guides and quick references for:
- Installation and setup
- C++ API usage
- Plugin configuration
- Connection to Python backend
- Troubleshooting

### `/implementation`
Implementation details including:
- Week-by-week completion summaries
- Technical implementation guides
- Verification and testing checklists
- Review summaries
