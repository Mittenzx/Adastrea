================================================================================
                           STATUS LIGHT EXAMPLES
================================================================================

SCENARIO 1: All Systems Operational (Normal State)
────────────────────────────────────────────────────────────────────────────────
  ● Python Process                    ● IPC Connection
  ● Python Bridge Ready               ● Backend Health
  ● Query Processing                  ● Document Ingestion

All lights are GREEN - system is fully operational and ready for use.

────────────────────────────────────────────────────────────────────────────────

SCENARIO 2: Python Process Not Running
────────────────────────────────────────────────────────────────────────────────
  ● Python Process                    ● IPC Connection
  ● Python Bridge Ready               ● Backend Health
  ● Query Processing                  ● Document Ingestion

Python Process, IPC, Bridge, and Backend Health are RED.
Query Processing and Ingestion are also RED as they depend on the bridge.
Action: Check Settings for correct Python path and restart.

────────────────────────────────────────────────────────────────────────────────

SCENARIO 3: Currently Processing Query
────────────────────────────────────────────────────────────────────────────────
  ● Python Process                    ● IPC Connection
  ● Python Bridge Ready               ● Backend Health
  ● Query Processing                  ● Document Ingestion

All lights GREEN, Query Processing shows "Active" status.
This is a temporary state while a query is being processed.

────────────────────────────────────────────────────────────────────────────────

SCENARIO 4: Document Ingestion Running (42% complete)
────────────────────────────────────────────────────────────────────────────────
  ● Python Process                    ● IPC Connection
  ● Python Bridge Ready               ● Backend Health
  ● Query Processing                  ● Document Ingestion

All lights GREEN except Document Ingestion which is YELLOW/ORANGE.
This indicates an active operation in progress.

────────────────────────────────────────────────────────────────────────────────

SCENARIO 5: IPC Connection Lost
────────────────────────────────────────────────────────────────────────────────
  ● Python Process                    ● IPC Connection
  ● Python Bridge Ready               ● Backend Health
  ● Query Processing                  ● Document Ingestion

Python Process may be GREEN but IPC Connection is RED.
This causes Bridge, Backend, Query, and Ingestion to be RED.
Action: Click "Reconnect" button or restart Python backend.

────────────────────────────────────────────────────────────────────────────────

SCENARIO 6: First Launch / Checking Status
────────────────────────────────────────────────────────────────────────────────
  ● Python Process                    ● IPC Connection
  ● Python Bridge Ready               ● Backend Health
  ● Query Processing                  ● Document Ingestion

All lights GRAY - status is being determined.
This is the initial state before first status check completes.

================================================================================

KEY:
  ● GREEN  - Component operational / ready
  ● YELLOW - Warning / operation in progress (like ingestion)
  ● RED    - Error / not functional / disconnected
  ● GRAY   - Unknown / checking status

The status lights automatically update every 0.5 seconds when viewing the
Dashboard tab, providing real-time feedback on system health.

For detailed troubleshooting, see STATUS_INDICATORS.md
