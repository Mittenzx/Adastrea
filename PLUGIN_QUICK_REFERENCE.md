## How to Enable Plugins

### Built-In Plugins:
1. Edit ‚Üí Plugins
2. Search for plugin name
3. Check "Enabled"
4. Restart editor

### Marketplace Plugins:
1. Purchase on [Fab.com](https://www.fab.com/)
2. Install via Epic Games Launcher ‚Üí Library ‚Üí Vault
3. Enable in project (Edit ‚Üí Plugins)
4. Restart editor
