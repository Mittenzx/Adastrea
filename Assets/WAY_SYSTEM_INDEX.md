## Key Concepts Reference

### Core Terms

- **Way** - Small specialized guild (50-1000 members) focused on specific industry
- **Network** - Micro-alliance of 2-5 Ways with shared Precepts (values)
- **Precept** - Core value that defines what a Way stands for (Craftsmanship, Honor, etc.)
- **Feat** - Legendary accomplishment that grants reputation with Ways
- **Spillover** - Reputation gain transferred to network members (20-70% typical)
- **Network Bonus** - Reputation multiplier for aligned Feats (1.5-2.5x typical)
- **Verse Score** - Player's total reputation with a network
- **Sector Council** - Practical governance body where Ways vote on taxes and policies

### Key Classes

- **UWayDataAsset** - Blueprint Data Asset defining a Way
- **UWayNetworkDataAsset** - Blueprint Data Asset defining a network
- **UVerseSubsystem** - Game Instance Subsystem managing networks globally
- **UVerse** - Actor Component tracking player reputation with Ways

### Key Functions

- **RegisterNetwork()** - Register a network with the subsystem
- **GetNetworksForWay()** - Find all networks a Way belongs to
- **GetNetworkVerseScore()** - Calculate player's network reputation
- **QualifiesForNetworkBonuses()** - Check if player meets threshold
- **RecordFeatWithNetworkEffects()** - Award Feat with spillover
